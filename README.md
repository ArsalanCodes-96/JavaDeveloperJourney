# 01 - Basics of Programming

##  What is Programming?

Programming is the process of giving instructions to a computer to perform specific tasks. These instructions are written in a programming language that both humans and computers can understand.

##  What is a Programming Language?

A programming language is a formal language used to write instructions (called programs) that tell a computer what to do. It acts as a bridge between humans and machines.

##  Why Do We Need Programming Languages?

- To communicate with computers.
- To automate repetitive tasks.
- To build software, websites, mobile apps, and games.
- To solve real-world problems using technology.

##  Types of Programming Languages

### 1. Machine Language (1GL)
- The lowest-level language.
- Consists of only `0`s and `1`s (binary code).
- Directly understood by the computer.
- Very fast but difficult for humans to write and understand.

### 2. Assembly Language (2GL)
- Uses mnemonic codes like `ADD`, `MOV`, and `SUB`.
- Easier than machine language.
- Requires an assembler to convert it into machine code.

### 3. High-Level Language (3GL)
- Uses English-like syntax.
- Easy to read, write, and maintain.
- Requires a compiler or interpreter.
- Examples: Java, C, C++, Python.

### 4. Fourth-Generation Language (4GL)
- Designed to be more user-friendly and require fewer lines of code.
- Commonly used for database queries and report generation.
- Example: SQL.

### 5. Fifth-Generation Language (5GL)
- Focuses on problem-solving using constraints and logic.
- Often used in Artificial Intelligence (AI) and expert systems.

##  Compiler vs Interpreter

| Compiler | Interpreter |
|----------|------------|
| Translates the entire program at once. | Translates and executes one line at a time. |
| Faster execution after compilation. | Slower execution. |
| Example: C, C++ | Example: Python, JavaScript |

##  Why Learn Java?

- Platform Independent ("Write Once, Run Anywhere").
- Object-Oriented Programming (OOP).
- Widely used in backend development, Android development, and enterprise applications.
- Strong community support and job opportunities.

##  Key Terms

- **Algorithm:** A step-by-step solution to a problem.
- **Program:** A set of instructions written to perform a task.
- **Source Code:** The human-readable code written by a programmer.
- **Compiler:** Converts source code into machine code.
- **Interpreter:** Executes code line by line.
- **Bug:** An error or flaw in a program.
- **Debugging:** The process of finding and fixing bugs.
